# Полный аудит проекта Archivist-Library (MGG Hub)

## Краткое описание проекта

**Archivist-Library** — русскоязычный сайт-энциклопедия и инструментарий для игры "Mutants Genetic Gladiators" (MGG). Проект представляет собой комплексную базу знаний с интерактивными симуляторами, калькуляторами и справочниками по всем аспектам игры: мутанты, скрещивания, материалы, рулетки, крафт, бинго и рейтинговые системы.

**Живой сайт:** https://www.archivist-library.com (CDN: https://cdn.archivist-library.com)

## Стек технологий

### Основной фреймворк
- **Astro 5** (SSG-фреймворк) — генерация статических страниц с серверным рендерингом
- **TypeScript** — типизация всего кодовой базы

### Интерактивные компоненты
- **Svelte 5** — реактивные UI-компоненты (симуляторы, калькуляторы, браузеры)
- **@astrojs/svelte** — интеграция Svelte в Astro

### Стилизация
- **Tailwind CSS 4** через Vite-плагин
- **@tailwindcss/typography** — плагин для типографики
- Кастомные CSS-переменные в `global.css`

### Инструменты сборки и разработки
- **Vite** — сборщик (встроен в Astro)
- **vite-plugin-raw** — импорт .txt файлов как строки
- **@tailwindcss/vite** — Tailwind интеграция

### Данные и обработка
- **XLSX** (xlsx) — парсинг Excel файлов (данные мутантов, тиров)
- **fast-xml-parser** — парсинг XML (настройки скрещивания)
- **Sharp** — оптимизация изображений

### Деплой и аналитика
- **@astrojs/vercel** — адаптер для деплоя на Vercel
- **@vercel/analytics** — аналитика посещений
- **@vercel/speed-insights** — метрики производительности

### Разработка
- **ESLint** с плагинами для Astro и Svelte
- **Prettier** — форматирование кода
- **tsx** — выполнение TypeScript скриптов
- **axios** — HTTP-запросы (для загрузки данных из Google Sheets)

### Паттерны компонентов

1. **Astro-компоненты** (.astro) — статические layout и страницы
2. **Svelte-компоненты** (.svelte) — интерактивный UI
3. **Гибридный подход**: Astro для статической оболочки, Svelte для динамических частей

### Управление состоянием

- **Локальное состояние Svelte** ($state, $derived) — для UI-состояния симуляторов
- **Props-передача** — данные от Astro к Svelte через client:load
- **Кэширование** — мемоизация вычислений в MutantsBrowser

### Стилизация

- **Темная тема** (#0d1117 фон, #c9d1d9 текст)
- **Кастомный шрифт** TT Supermolot Neue
- **Фоновое изображение** с фильтрами (blur, saturate, brightness)
- **Адаптивный дизайн**: mobile-first с breakpoints:
  - Mobile: < 768px
  - Tablet: 768-1023px
  - Desktop: 1024px+
  - 2K/4K: 1921px+, 3840px+ (автомасштабирование)

## Ключевые механизмы и нестандартные решения

### 1. Алгоритм скрещивания мутантов
**Файл:** `src/lib/breeding/breeding.ts`

Многоуровневая система предсказания потомства:
- **Секретные рецепты** — приоритетные комбинации (хардкод в `secretCombos.ts`)
- **Генетический алгоритм** — расчет возможных генных комбинаций
- **Фильтрация по типам** — проверка доступности мутантов для скрещивания
- **Весовые шансы** — учет вероятностей выпадения

### 2. Система гача/реактора
**Файл:** `src/lib/reactor-gacha.ts`

Реалистичная симуляция генераторов наград:
- **Взвешенные награды** — каждая награда имеет вес (odds)
- **Прогресс-система** — отслеживание собранных наград
- **Комплиментарная награда** — бонус за полный сбор
- **Динамический пул** — изменение шансов при выпадении редких наград

### 3. Многоуровневый крафт
**Файл:** `src/lib/craft-simulator.ts`

4 типа заводов с уникальными механиками:
- **Metal Factory** — переплавка звезд (5 одинаковых → 1 следующий ранг)
- **Transformatron** — трансформация сфер
- **Laboratory** — создание предметов
- **Black Hole** — экспериментальные рецепты

Каждый завод имеет:
- Систему "инцентива" (бонусные награды за время крафта)
- Симуляцию до 1,000,000 крафтов
- Детальную статистику выпадений

### 4. Калькулятор статов
**Файл:** `src/lib/stats/unified-calculator.ts`

Точный расчет характеристик мутантов с учетом:
- Базовых статов (lvl1, lvl30)
- Сфер (орбов) — тип, уровень, бонусы
- Спецсфер и модификаторов
- Скоростных сфер

### 5. Система бинго
**Файл:** `src/pages/bingo.astro`

Сложная структура данных:
- **Двумерные сетки** — мутанты в rows × cols
- **Награды за строки/столбцы** — линейные бонусы
- **Награда за завершение** — главный приз
- **Заголовки** — генетические иконки и тематические изображения
- **Под-бинго** — автоматическое разделение больших сеток

### 6. CDN-система текстур
**Файл:** `src/lib/texture-cdn.ts`

Динамическая генерация URL изображений:
- Основной CDN: `cdn.archivist-library.com`
- Fallback на локальные пути
- Оптимизация через Sharp (webp формат)
- Маппинг скинов мутантов

### 7. Адаптивный дизайн с 2K/4K поддержкой
**Файл:** `src/layouts/BaseLayout.astro`

Уникальное решение для high-DPI дисплеев:
- CSS-медиа-запросы для масштабирования
- Автоматическое увеличение шрифтов (125% для 2K, 150% для 4K)
- Динамические max-width контейнеров
- Сохранение пропорций интерфейса

### 8. Гибридная маршрутизация
- **SSG + SSR**: большинство страниц статические (`prerender: true`)
- **Динамические страницы**: `/top-evo` загружает данные из Google Sheets
- **Клиентская навигация**:.prefetch для мгновенных переходов

### 9. Система поиска и фильтрации
**Файлы:** `src/lib/search-normalize.ts`, `src/lib/mutant-sort.ts`

- **Нормализация поиска** — игнорирование регистра, символов
- **Сортировка по генам** — группировка мутантов по генетическим типам
- **Мемоизация** — кэширование результатов фильтрации

### 10. Безопасность и производительность
**Файл:** `vercel.json`

- **CSP (Content Security Policy)** — строгие правила загрузки ресурсов
- **X-Frame-Options: DENY** — защита от clickjacking
- **HSTS** — принудительное HTTPS
- **Vercel Analytics** — мониторинг производительности

## Данные проекта

### Структура данных
- **Мутанты**: `src/data/mutants/mutants.json` (59,120 строк) — полная база с характеристиками
- **Скины**: `src/data/mutants/skins.json` — альтернативные модели мутантов
- **Бинго**: `src/data/bingos.json` (5,987 строк) — все бинго-сетки
- **Материалы**: `src/data/materials/*.json` — сферы, бустеры, здания, зоны
- **Симуляторы**: `src/data/simulators/**/*.json` — настройки гача, рулеток, крафта

- **Google Sheets**: загрузка топа эво через `google-sheets-loader.ts`
- **Локализация**: `localisation_ru.txt`, `localisation_en.txt`

**Archivist-Library** — это зрелый, архитектурно продуманный проект с:
- Чистым разделением статики (Astro) и динамики (Svelte)
- Точной игровой логикой (симуляторы с реальными шансами)
- Масштабируемой структурой данных

Проект представляет собой полноценную энциклопедию для сообщества MGG с интерактивными инструментами, точно воспроизводящими игровую механику.
