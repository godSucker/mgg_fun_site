# Мобильная адаптация MGG Hub

## Обзор

Сайт MGG Hub теперь полностью адаптирован для мобильных устройств с использованием современного responsive design подхода.

## Ключевые особенности

### 1. Гамбургер-меню

На мобильных устройствах (< 768px) горизонтальная навигация заменяется на компактную иконку гамбургер-меню в правом верхнем углу. При нажатии открывается выдвижная боковая панель с полным списком разделов сайта.

**Функционал:**
- Плавная анимация открытия/закрытия
- Затемненный фон (overlay) для фокусировки на меню
- Автоматическое закрытие при выборе пункта
- Блокировка прокрутки основного контента при открытом меню

### 2. Адаптивная сетка карточек

Карточки разделов автоматически адаптируются под размер экрана:
- **Mobile (< 640px):** 1 колонка
- **Tablet (640px - 1023px):** 2 колонки
- **Desktop (≥ 1024px):** Автоматическая сетка (3-4 колонки)

### 3. Оптимизированная типографика

Размеры шрифтов и отступы масштабируются в зависимости от размера экрана для обеспечения оптимальной читаемости.

### 4. Touch-friendly интерфейс

Все интерактивные элементы имеют достаточный размер для удобного нажатия пальцем (минимум 44x44px по рекомендациям Apple).

## Технические детали

### Breakpoints

```css
/* Mobile */
@media (max-width: 767px) { ... }

/* Tablet */
@media (min-width: 768px) and (max-width: 1023px) { ... }

/* Desktop */
@media (min-width: 1024px) { ... }
```

### Измененные компоненты

1. **BaseLayout.astro** - основной layout с мобильным меню
2. **index.astro** - адаптивная главная страница
3. **CardGrid.astro** - адаптивная сетка карточек
4. **Card.astro** - адаптивные карточки
